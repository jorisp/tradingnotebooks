# Jupyter notebooks for data driven investors

<ul>
<li>[Indices Jupyter notebook](indices.ipynb) shows how you can use seaborn to visualise a yearly return heatmap for a number of stock indices ( BEL20, AEX,DJIA, S&P 500,Euro Stoxx 50) and then compares the yearly returns of these indices against the SPDR MSCI ACWI IMI ETF.
<li>[AAPL trading rule using SMA](AAPL_SMA.ipynb) shows how you can create a trading rule to buy when SMA1>SMA2 (the shorter SMA is above the longer SMA) and how to do a vectorized backtesting</li>
</ul>
